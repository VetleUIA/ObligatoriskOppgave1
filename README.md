# Universitetssystem – C# Console Application

## Hvordan kjøre programmet

1. Last ned prosjektet
2. Åpne i Rider eller Visual Studio
3. Kjør Program.cs

## Funksjonalitet
- Opprette kurs
- Melde studenter på kurs
- Søke etter kurs
- Bibliotek med utlån og retur
- Lånehistorikk
- Printe kurs og deltakere
